# Open Development Vietnam (data-vietnam-opendevelopmentmekong-net)

Open Development Vietnam is a organization open-data portal for Vietnam running **CKAN**, exposing a standardized catalog API over ~13,484 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-vietnam-opendevelopmentmekong-net/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-vietnam-opendevelopmentmekong-net-api-evangelist&utm_content=repo)

## Type
- **kind:** organization  ·  **software:** CKAN  ·  **version:** 2.9.9
